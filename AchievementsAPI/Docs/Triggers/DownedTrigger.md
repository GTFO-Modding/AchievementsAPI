# DownedTrigger (ID: PlayerDown)
A trigger that is activated when the local player downs.

## Data
None.

# Example JSON

Player must down 3 times
```json
{
    "ID": "PlayerDown",
    "Count": 3,
    "ConditionOverrides": {
        "HasOverrides": false,
        "AdditionalConditions": []
    }
}
```
